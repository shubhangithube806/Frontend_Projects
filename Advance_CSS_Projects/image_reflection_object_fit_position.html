2.The object-fit Property

Specifies how an img or video should be resized to fit its container

No-object-fit:

object-fit: fill(this is default)

The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit

object-fit: contain;

The image keeps its aspect ratio, but is resized to fit within the given dimension

object-fit: cover;

the image is scaled down to the smallest version of none or contain

object-fit: scale-down;

the image is scaled down to the smallest version of none or contain

object-fit: none

The image is not resized
