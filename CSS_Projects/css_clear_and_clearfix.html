CSS Clear And Clearfix

1.Clear

When clearing floats, you should match the clear to the float: If an element is floated to the left, then you should clear to the left. Your floated element will continue to float, but the cleared element will appear below it on the web page.

i.Without clear

div4
div5 - Notice that div5 is after div4 in the HTML code. However, since div4 floats to the left, the text in div5 flows around div1.


ii.With clear

div6
div7 - Here, clear: left; moves div7 down below the floating div6. The value "left" clears elements floated to the left. You can also clear "right" and "both".
