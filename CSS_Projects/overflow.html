CSS Overflow

1.overflow:visible

visible - Default. The overflow is not clipped. The content renders outside the element's box

By default, the overflow is visible, meaning that it is not clipped and it renders outsidethe element's box: The overflow property only works for block elements with a specified height.


2.overflow:hidden

hidden - The overflow is clipped, and the rest of the content will be invisible
