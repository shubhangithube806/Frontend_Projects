*Relative Lengths*

1.With 'em'

"Relative to the font-size of the element (2em means 2 times the size of the current font)"

These paragraphs have a calculated line-height of: 2x16px = 32px.

These paragraphs have a calculated line-height of: 2x16px = 32px.

The font size of the div element is set to 30px. The span element inside the div element has a font-size of 0.5em, which equals to 0.5x30 = 15px.

2.With 'rem'

"Relative to font-size of the root element"

The font-size of this document is 16px.

The font-size of this div element is 3rem. font size of this span is 2rem

3.With Viewport Width 'vw'

Relative to 1% of the width of the viewport*

Welcome

Resize the width of the browser window to see how the font-size of h1 changes.

1vw = 1% of viewport width.

4.With Viewport Height 'vh'

Relative to 1% of the height of the viewport*

Welcome

Resize the height of the browser window to see how the font-size of h1 changes.

1vh = 1% of viewport height.

5.With 'vmin'

Relative to 1% of viewport's* smaller dimension

Hello

Resize the browser window (both width and height) to see how the font-size of h1 changes.

1vmin = 1vw or 1vh, whichever is smaller.

6.With 'vmax'

Relative to 1% of viewport's* larger dimension

Hello

Resize the browser window (both width and height) to see how the font-size of h1 changes.

1vmax = 1vw or 1vh, whichever is larger.

The vmax unit is not supported in Internet Explorer or Safari 6.1 and earlier versions.

7.With '%'

Relative to the parent element

The font-size of this document is 16px.

The font-size of this div element is 150%.

The % unit sets the font-size relative to the current font-size.
